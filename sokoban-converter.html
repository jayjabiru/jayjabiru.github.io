How to Use
1. In sokobanonline.com builder, finish your level and click Save.
2. Open your browser console (F12) and go to the Network tab, or use the Actions → Export button if available. Alternatively, use the level's URL ID and the site's API endpoint.
3. Paste the raw JSON below and click Convert.
4. Copy the output map string and paste it into the next null slot in levels.js.
